# react-native-purchasely-amazon

Purchasely Amazon In-App Purchases dependency

## Installation

```sh
npm install react-native-purchasely-amazon
```

## Usage

```js
import { multiply } from "react-native-purchasely-amazon";

// ...

const result = await multiply(3, 7);
```
